## Overview

OrbisSDR is a Windows desktop application for browsing SDR receiver sites inside a native wrapper.

This repository includes two compiled installers:

- `OrbisSDR Setup 0.1.1.exe` — single-window app
- `OrbisSDR_Multi Setup 0.1.1.exe` — multi-window app

## Installation

Use the installer executable for the version you want and follow the setup wizard.

### Installer files

- `OrbisSDR Setup 0.1.1.exe`
- `OrbisSDR_Multi Setup 0.1.1.exe`

## Notes

- These instructions assume you already have the compiled installers.
- You do not need to install Node.js or npm to use the setup executables.
- Discord Rich Presence is optional and can be enabled via `DISCORD_CLIENT_ID` when running the app. (by default its always on)
- If you want source or rebuild instructions, you can ask for the application source, but they are not required for using the installers.
